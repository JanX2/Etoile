 Navigation:
    Tabs, the toolbar, contextual menus and the address book, all help to aid navigation in iTerm. If don't use these features, give them a look, because they can be extremely helpful.
Contexual Menu:
  • Has access to your Address Book!!
  • Has access to the Configure panel
  • Has access to clear the buffer
  • Selected text can sent to either your web browser, your email program, saved to file or copied.
      Text sent to your mail program should be an email address
      Text sent to your browser should be a URL
Address Book:
  • The Address Book organizes preset terminal settings.
  • Instead of modifying the configure panel or preference panel, you can save seperate settings for a session
  • For the command you can specify an command, doesn't have to be a shell.
  • The Address Book can be accessed from a contextual menu, the Tool Bar or from the menu: Window -> Address Book (Command + Shift + A)
