 iTerm:
    iTerm is a full featured terminal emulation program written for OS X using Cocoa. It supports language encodings, VT100/ANSI/XTERM emulation and many convenient GUI features. iTerm was merged from CTerminal, TerminalX and JTerminal. A large part of the original code is rewritten from the JTerminal source to implement more features and to run more efficiently. The current version is still in beta stage.
Features:
  • Native Cocoa application that supports OS X, Jaguar.
  • VT100 emulation, faster than the Terminal.app that comes with OS X
  • Multi-Tab within one window
  • Xterm compatibility
  • Transparent windows
  • Address book
  • xterm emulation makes running most unix programs comfortable (To be exact, xterm emulation is more rxvt like).
  • Anti-idle function that avoids disconnection because of no activity
  • Supported language encodings including special support for:
    • Chinese: GB/EUC-CN and BIG5
    • Japanese: EUC-JP and SJIS
    • Korean: EUC-KR
