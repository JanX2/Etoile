 System Requirements:
    Mac with Mac OS X 10.2 (Jaguar)
Install:
    Mount the image file and copy iTerm to anywhere you want. If you have an older version, be sure it's either over-written or completely gone (ie. not in trash). OS X sometimes has weird behavior about that.
Tips:
  • Starting from version 0.5.5, iTerm uses a different approach to support all encodings. Thus the old encoding settings are lost and they have to be manually reset.
  • If you are originally running CTerminal and wish to keep your old address book, you need to rename your "CTerminal Address Book" file as "iTerm Address Book".
  • To get precise character position matching, please choose Osaka-regular mono font. This is the only font that I find to be really even spaced.
  • The address book is stored in ~/Library/Application SupportIt's not encrypted, so be careful of your sensitive information
  • In the command field in Quick Run and Address Book windows, you can type in arguments for the command you wish to run. For example, if you want to run bash with existing environment settings, use bash --login
  • To be able to transfer CJK characters in a telnet session, you might want to use -8 option. eg: telnet -8 fsbbs.org 3000
