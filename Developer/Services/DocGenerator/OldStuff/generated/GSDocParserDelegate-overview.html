Presentation

This class is a NSXMLParser delegate, which extracts informations from a gsdoc file.

Once extracted, an HTML file can be generated using a template.

Template File

A template is necessary to generate the final HTML file. Such template is a normal HTML file, containing two special comments:

  • <!-- gsdoc-header --> : insert the generated header
  • <!-- gsdoc-methods --> : insert the generated methods list
  • <!-- gsdoc-files --> : insert the content of the file “filelist.html” (if the file is present in the GSDoc source directory)
